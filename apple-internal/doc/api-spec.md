# API Spec

## API Usage Examples
### Get Application Log
```bash
curl -N -u appleconnect_user_name:dummy_password <end_point>/skatev1/spark/c01-56bd5502e9b14501afc5123b73213b17/log
```

### Kill and Delete Application:
```bash
curl -u appleconnect_user_name:dummy_password -X DELETE <end_point>/skatev1/spark/c01-56bd5502e9b14501afc5123b73213b17
```
